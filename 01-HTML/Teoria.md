# HTML5
## Hypertext Markup Language

- Estructura de la página
- Lenguaje de Marcas
- Estructura Fundamental de la Web
- Navegador Web + IDE

## Funcionamiento

Cada página web tiene una estructura básica, que es la que se encuentra en el archivo HTML.

- Etiquetas
- Atributos
- Elementos
